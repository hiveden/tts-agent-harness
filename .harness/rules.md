# TTS Rules

P1 normalize 和 P4 Claude 都必须遵守这些规则。人工维护，harness 只读。

## 英文处理
- 句首英文品牌名前加 `.`（`.AutoGPT 十八万星标`）
- 句中英文保持原样，不转中文，不音译人名
- 英文连字符改空格（`yoyo-evolve` → `yoyo evolve`）
- OpenAI 句中加空格（`Open AI`）

## 数字处理
- 年份转中文（`2024年` → `二零二四年`）
- 超4位纯数字转中文（`60000` → `六万`）
- 百分比转中文（`53%` → `百分之五十三`）
- 数字范围 dash 转"到"（`28-35K` → `28到35K`）
- 带中文单位的数字保留原样（`630行`）

## TTS 模型
- 默认速度 1.15x
- 中文模型能读大部分英文品牌名，不需要音译

## P4 修复约束
- 优先调断句/格式，不改原始含义
- 同音字替换不算错误（的/地/得）
- 只修复 high severity，low 自动放行
